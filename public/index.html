WALL/THERAPY REST API

This is the landing page the WALL/THERAPY REST API. You can query this API using the end points /murals and /artists for information on WALL/THERAPY murals and artists.

For an example of how this API can be used by an application, see the React based frontend I developed for this final project at https://walltherapy-frontend.netlify.com/

Admin Panel:

The Strapi.io admin panel at https://walltherapy-strapi.herokuapp.com/admin allows admin users to add new Mural and Artists to the API's database with a online editor. Use the username admin and the password adminpassword to login to the admin panel. Check out the Strapi.io documentation for more informaton on using the framework and editor, but to start check out the 'Content Types' menu on the left side see how the current databases items are created.
